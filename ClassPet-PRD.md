# PRD: 클래스펫 (ClassPet)

> 담임교사를 위한 다마고치 스타일 종합 학급 경영 플랫폼

---

## 1. 프로젝트 개요

### 1.1 한 줄 요약
**"시간표, 진도, 사회정서를 통합 관리하며 학생 성장을 귀여운 펫으로 시각화하는 담임교사용 웹앱"**

### 1.2 배경 및 동기
- [그라운드카드](https://growndcard.com/)의 사회정서 기반 학급 경영 시스템에서 영감
- 담임교사로서 시간표, 진도, 학생 관리를 한 곳에서 처리할 필요
- 단순한 상벌점이 아닌, 다마고치처럼 함께 성장하는 경험 제공
- 기존 서비스들은 기능이 분산되어 있거나 유료

### 1.3 목표 사용자
- **주 사용자**: 초등학교 담임교사
- **간접 사용자**: 담당 학급 학생들 (약 25명 내외)

### 1.4 핵심 가치
```
📚 학습 관리 + 💝 사회정서 + 🐾 성장의 즐거움
```

### 2.2 🐾 펫 시스템 (사회정서 + 동기부여)

#### 2.2.1 펫 기본 구조
```
[학생 1명] = [펫 1마리]

펫 상태:
├── 이름 (학생이 직접 설정)
├── 종류 (강아지/고양이/토끼/햄스터/병아리)
├── 레벨 (1~10)
├── 경험치 (EXP)
├── 성장 단계 (알 → 아기 → 성장 → 성체)
└── 현재 기분 (최근 활동에 따라)
```

#### 2.2.2 성장 단계 (4단계)
| 단계 | 이름 | 필요 EXP | 외형 변화 |
|------|------|----------|-----------|
| 1단계 | 알 | 0 | 🥚 알 |
| 2단계 | 아기 | 50 | 🐣 아기 동물 |
| 3단계 | 성장 | 150 | 🐕 청소년 |
| 4단계 | 성체 | 300 | 🦮 완전 성장 |

#### 2.2.3 칭찬 카테고리 (2022 개정교육과정 기반)
| 카테고리 | 아이콘 | EXP | 설명 |
|----------|--------|-----|------|
| 자기관리 | 🎯 | +10 | 준비물, 과제, 시간 관리 |
| 지식정보 | 📚 | +10 | 학습 태도, 발표, 질문 |
| 창의적사고 | 💡 | +15 | 새로운 아이디어, 문제해결 |
| 심미적감성 | 🎨 | +10 | 예술활동, 감상, 표현 |
| 협력적소통 | 🤝 | +10 | 모둠활동, 경청, 배려 |
| 공동체 | 🏠 | +10 | 규칙, 봉사, 리더십 |

#### 2.2.4 빠른 칭찬 입력
```
[학생 선택] → [카테고리 탭] → 완료!

예: 김철수 탭 → [🤝협력] 탭 → "+10 EXP" & 펫 반응
```

### 2.3 💝 사회정서 관리

#### 2.3.1 감정 체크인 (선택적)
```
아침 조회 시:
"오늘 기분은 어때요?"

😊 좋아요  😐 보통이에요  😢 안좋아요  😴 피곤해요
```

#### 2.3.2 관계 메모
```
학생별 특이사항/상담 기록:
├── 김철수: 1/15 - 친구 관계 고민 상담
├── 이영희: 1/18 - 발표 불안 있음, 격려 필요
└── 박민수: 1/20 - 오늘 기분 안좋음 (가정문제?)
```

#### 2.3.3 알림/주의 표시
```
⚠️ 주의가 필요한 학생:
├── 😢 이영희 - 3일 연속 "안좋아요" 선택
└── 📉 박민수 - 최근 2주간 EXP 획득 없음
```

## 8. 성공 지표

### 사용성
- 칭찬 입력까지 **2탭 이내**
- 오늘 수업 확인까지 **1초 이내**
- 한 학기 **꾸준히 사용**

### 교육적 효과
- 학생들이 펫 성장을 **기대함**
- 긍정적 행동의 **자발적 증가**
- 학급 분위기 **개선**
