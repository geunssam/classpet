# PRD: 체육 성장펫 (PE Growth Pet)

> 체육 수업을 위한 다마고치 스타일 학급 경영 플랫폼

---

## 1. 프로젝트 개요

### 1.1 한 줄 요약
**"체육 수업에서 학생들의 긍정적 행동을 귀여운 동물 캐릭터 성장으로 시각화하는 웹앱"**

### 1.2 배경 및 동기
- [그라운드카드](https://growndcard.com/)의 사회정서 기반 학급 경영 시스템에서 영감
- 체육 전담 교사로서 여러 학급을 효율적으로 관리할 필요
- 학생들에게 즉각적인 시각적 피드백을 통한 동기 부여 제공
- 단순한 상벌점이 아닌, 성장 과정을 함께 보는 경험 제공

### 1.3 목표 사용자
- **주 사용자**: 체육 전담 교사 (선생님)
- **간접 사용자**: 초등학생 (1~4학급, 약 100명 이하)

### 1.4 핵심 가치
```
도전 > 규칙 > 협동 > 기능
```
- **도전**: 못해도 끝까지 노력하기, 새로운 것 시도하기
- **규칙**: 준비물 챙기기, 집합 잘하기, 정리정돈
- **협동**: 친구 도와주기, 팀 귀속감, 협력 플레이
- **기능**: 달리기, 던지기, 체력 등 실력 향상

## 2. 핵심 기능

### 2.1 펫 시스템 (다마고치 스타일)

#### 2.1.1 펫 기본 구조
```
[학생 1명] = [펫 1마리]

펫 상태:
├── 이름 (학생이 직접 설정 또는 교사 설정)
├── 레벨 (1~10)
├── 경험치 (EXP)
├── 성장 단계 (알 → 아기 → 성장 → 성체)
└── 현재 기분 (최근 활동에 따라 변화)
```

#### 2.1.2 성장 단계 (4단계)
| 단계 | 이름 | 필요 EXP | 외형 |
|------|------|----------|------|
| 1단계 | 알 | 0 | 🥚 알 상태 |
| 2단계 | 아기 | 50 EXP | 🐣 아기 동물 |
| 3단계 | 성장 | 150 EXP | 🐕 청소년 동물 |
| 4단계 | 성체 | 300 EXP | 🦮 완전 성장 |

#### 2.1.3 동물 종류 (학급/학생별 선택)
- 강아지 🐕 → 🦮
- 고양이 🐱 → 🐈
- 토끼 🐰 → 🐇
- 햄스터 🐹 → 🐿️
- 병아리 🐣 → 🐓

### 2.2 포인트 시스템

#### 2.2.1 칭찬 카테고리
| 카테고리 | 아이콘 | 기본 EXP | 설명 |
|----------|--------|----------|------|
| 도전 | ⭐ | +15 | 어려운 것에 도전, 끝까지 노력 |
| 규칙 | 📋 | +10 | 준비물, 집합, 정리정돈 |
| 협동 | 🤝 | +10 | 팀워크, 친구 도움 |
| 기능 | 💪 | +5 | 운동 실력 향상 |

#### 2.2.2 빠른 입력 방식
```
수업 중 시나리오:

[학급 선택] → [학생 선택(복수 가능)] → [카테고리 탭]

예시: 3학년 2반 → 김철수, 이영희 선택 → [도전⭐] 탭
      → 두 학생 모두 +15 EXP
```

### 2.3 학급 관리

#### 2.3.1 멀티 클래스 구조
```
선생님 계정
├── 3학년 1반 (25명)
├── 3학년 2반 (23명)
├── 4학년 1반 (26명)
└── 4학년 2반 (24명)
```

#### 2.3.2 학급별 뷰
- **펫 농장 뷰**: 학급 전체 펫을 한눈에 (귀여운 농장/마을 배경)
- **리스트 뷰**: 학생 목록 + 현재 EXP + 레벨
- **포인트 입력 뷰**: 빠른 포인트 부여용

### 2.4 시각화 대시보드

#### 2.4.1 학급 전체 뷰 (펫 농장)
```
┌─────────────────────────────────────────┐

## 3. 사용 시나리오

### 3.1 수업 중 실시간 활용
```
상황: 3학년 2반 피구 수업 중

1. 태블릿으로 앱 열기
2. "3학년 2반" 학급 선택
3. 경기 중 협동 잘하는 학생 발견
4. 해당 학생 이름 탭 → "협동🤝" 버튼
5. 즉시 "+10 EXP" 팝업 & 펫 반응 애니메이션
6. 수업 마무리 시 "오늘의 MVP" 학생에게 추가 포인트
```

### 3.2 수업 후 기록 정리
```
상황: 교무실에서 오늘 수업 정리

1. 앱 열기 → 오늘 수업한 학급들 확인
2. 각 학급별로 기억나는 학생들에게 포인트 부여
3. "일괄 입력" 모드로 여러 학생 한번에 처리
4. 메모 기능으로 특이사항 기록
```

### 3.3 학생들에게 보여주기
```
상황: 수업 시작 시 동기부여

1. TV/빔프로젝터에 "펫 농장" 화면 띄우기
2. "지난 주 가장 많이 성장한 펫 Top 3" 발표
3. 레벨업한 펫 축하 애니메이션
4. 오늘의 미션 안내 (예: "협동 포인트 많이 받기")
```

## 4. 기술 스택 및 구조

### 4.1 기술 스택 (바이브 코딩 친화적)
```
프론트엔드:
├── HTML5 + Vanilla JavaScript
├── Tailwind CSS (스타일링)

## 8. 성공 지표

### 정량적 지표
- 수업 중 포인트 입력까지 **3탭 이내**
- 학급 전환까지 **2초 이내**
- 한 학기 동안 **꾸준히 사용**

### 정성적 지표
- 학생들이 펫 성장을 **기대하고 기다림**
- 체육 수업 시작 시 **"오늘 펫 보여주세요!"** 요청
- 긍정적 행동의 **자발적 증가**
